# 🛠 B站评论 & 消息管理工具

<BilibiliCommentManager/>

## 🌟 功能特点

| 功能 | 描述 | Emoji |
|------|------|-------|
| 管理回复 | 查看并管理自己发出的评论回复 | 📝 |
| 管理点赞 | 查看收到的评论点赞 | ❤️ |
| 批量删除 | 快速清理不想要的评论和消息 | 🗑️ |

## 📝 使用说明

### 1️⃣ 获取必需的 Cookie

要使用本工具，需要从 B站网页中获取以下三项 Cookie：

- **SESSDATA**  
- **bili_jct**  
- **DedeUserID**  

#### 获取步骤：

1. 打开 B站网页版，按下 **F12** 打开开发者工具  
2. 切换到 **“应用/存储 (Application/Storage)”**  
3. 找到 **“Cookies → https://www.bilibili.com”**  
4. 找到以下字段并复制对应值：

| Cookie 名称 | 说明 | Emoji |
|------------|-----|-------|
| SESSDATA  | 登录会话凭证 | 🔑 |
| bili_jct  | CSRF 验证用 token | 🛡️ |
| DedeUserID | 用户 ID | 🆔 |


5. 在工具页面粘贴到 **Cookie 输入框**：

SESSDATA=你的SESSDATA值; bili_jct=你的bili_jct值; DedeUserID=你的DedeUserID值

> ⚠️ 注意：一定要用英文分号 `;` 分隔三项 Cookie


### 2️⃣ 删除评论与消息

1. ✅ 选择 **要删除的评论或消息**  
2. 🗑️ 点击 **“删除选中项”** 或 **“批量删除”**  
3. ⚡ 工具会自动删除对应评论及相关的推送消息  

> 删除操作完全在本地执行，不上传数据，安全可靠。

### 3️⃣ 小提示

- 💡 使用前建议先备份重要评论  
- ⏳ 每次登录后 Cookie 有效期有限，过期需重新获取  
- ❌ 批量删除操作不可恢复，请谨慎操作
