## 学习路径

1. **基础阶段**：掌握 Java 基础语法和面向对象
2. **进阶阶段**：学习集合、多线程、JVM
3. **框架阶段**：学习 Spring、MyBatis 等框架
4. **项目阶段**：通过实际项目积累经验
5. **深入阶段**：学习分布式、微服务等高级技术
